## üìù Contribution Templates

We use standardized templates to maintain clarity and efficiency in our development workflow.

### 1. Issue Templates
Used when creating a new issue in any RepublicOfGamja repository.
* **Bug Report:** Structured template for reporting errors and steps to reproduce.
    * [View Template](ISSUE_TEMPLATE/bug_report.md)
* **Feature Request:** Template for proposing new ideas, including problem statements and proposed solutions.
    * [View Template](ISSUE_TEMPLATE/feature_request.md)

### 2. Pull Request Template
Ensures all PRs adhere to our standards by providing a checklist for testing, documentation, and commit convention adherence.
* [PULL_REQUEST_TEMPLATE.md](PULL_REQUEST_TEMPLATE.md)
